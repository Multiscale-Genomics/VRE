Cytospace. Network visualization and analysis
by MuGVRE development team

Cytospace is an open source bioinformatics software platform for visualizing molecular interaction networks and integrating with gene expression profiles and other state data. It includes Cytoscape.js, a graphsâ€™ browser for analyse and visualise networks in Java Script environments

Features:

  • Represents any network data model, like sequencing expression data, biomolecular interaction networks, etc.
  • Fully interative and responsive
  • Quick rendering images of graphs on Node.js with Cytosnap
  • Compatible with modern browsers, Node.js, R language, jQuery, npm

More information: http://js.cytoscape.org
