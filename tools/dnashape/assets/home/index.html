DNAShapeScan
by Remo Rohs (USC), Marco Pasi (University of Nottingham)

Predict binding sites for a protein on a given genome segment by using DNAShape to include information on the shape of DNA.

DNAShapeScan allows the user to create a DNA-binding affinity predictor for a target protein by integrating the commonly used, sequence-only models with information on the shape of DNA. The resulting predictor is used to identify binding sites on a genome. The resulting annotation can be viewed as a track in genome browsers.
