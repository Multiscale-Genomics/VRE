MC-DNA is a tool to rapidly create static and dynamic B-DNA conformations of a sequence of interest. With the use of a Monte Carlo algorithm this tool runs up to 50x faster than conventional Molecular Dynamics providing similar accuracy. MC-DNA provides a three-dimensional all-atom representation of the DNA structure with the underlying sequence of interest.

Two options are available within MC-DNA:

  • 'Create Structure' creates a DNA structure in equilibrium ground state conformation.
  • 'Create Trajectory' creates a trajectory of a selected amount of snapshots of the sequence of interest.

To provide a more in-depth analysis both outputs from 'Create Structure' and 'Create Trajectory' can also serve as an input for the tool NAflex.
