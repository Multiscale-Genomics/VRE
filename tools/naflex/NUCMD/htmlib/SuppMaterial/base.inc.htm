BIGNASim is a comprehensive database system and analysis portal holding nucleic acids MD trajectories and analyses. The database allows direct access to trajectory data, information about the structure and MD simulation metadata, together with a complete set of analyses performed. Additionally, the database is provided with a flexible NAFlex-based engine allowing users to perform their own analysis pipelines.
