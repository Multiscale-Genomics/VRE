Presentation

 BIGNASim is a comprehensive database system and analysis portal holding nucleic acids MD trajectories and their analyses. The database allows direct access to trajectory data, information about the structure and MD simulation metadata, together with a complete set of analyses performed. Additionally, the database is provided with a flexible NAFlex-based engine allowing users to perform their own analysis pipelines.

 BIGNASim uses state-of-the-art technology to deal with BigData management. Two well-known noSQL databases (Cassandra and Mongo) are used to store trajectory coordinates and analyses, respectively. These noSQL databases are queried using REST web services and php drivers.

 To facilitate the access to all information stored, we built a web portal on top of the database system, allowing several ways of accessing the data.
