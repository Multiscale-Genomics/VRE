Chromatin Dynamics provides a user friendly way to create individual 'beads-on-a-string' like representations of a chromatin fiber. The user decides the sequence of the linker DNA and where the nucleosomes are inserted which gives the user full control over designing his own individual chromatin fiber.

There are two possibilities to use Chromatin Dynamics:

In "Chromatin Dynamics from sequence and nucleosome positions" the user provides his own sequence and nucleosome position file. The option 'Create Structure' creates a ground state structure of the chromatin conformation. 'Create Trajectory' creates a trajectory of possible chromatin conformations. This option is recommended if average properties of the fiber want to be obtained. Note: A trajectory can only be created if the ground state structure does not physically overlap.

"Chromatin Dynamics from NucleR" creates a ground state structure of a selected genomic region based on the nucleosome positions determined by NucleR of the Nucleosome Dynamics workflow.
